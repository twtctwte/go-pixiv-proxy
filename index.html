**Tawerのpixiv反代** # 简介 Tawerのpixiv反代 ## 使用 ### 替换 i.piximg.net 为 pho.tawertwtc.xyz https://i.pximg.net/img-original/img/2022/05/21/22/35/46/98505703_p0.jpg 替换为 http://pho.tawertwtc.xyz/img-original/img/2022/05/21/22/35/46/98505703_p0.jpg ### url 后接 pid http://pho.tawertwtc.xyz/98505703 ## 图片质量设置 针对 **url 后接 pid** 的使用方法，提供了图片质量选择的参数 使用方法：添加参数 `t` 支持 - original - regular - small - thumb - mini 例子: http://pho.tawertwtc.xyz/98505703?t=original [](http://pho.tawertwtc.xyz/98505703?t=original) ## 代理 api http://pho.tawertwtc.xyz/api/pid ## 其他示范用例 ``` 1. http://pho.tawertwtc.xyz/$path - http://pho.tawertwtc.xyz/img-original/img/0000/00/00/00/00/00/12345678_p0.png 2. http://pho.tawertwtc.xyz/$pid[/$p][?t=original|regular|small|thumb|mini] - http://pho.tawertwtc.xyz/12345678 (p0) - http://pho.tawertwtc.xyz/12345678/0 (p0) - http://pho.tawertwtc.xyz/12345678/1 (p1) - http://pho.tawertwtc.xyz/12345678?t=small (small image) ```
